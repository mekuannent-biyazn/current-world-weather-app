# 🌍 Current World Weather App

A real-time weather application that displays live global weather data with interactive location search and map visualization.

---

## ✨ Features

- 🌦️ Real-time global weather data
- 🔎 City-based weather search
- 🗺️ Interactive map visualization
- 🌡️ Temperature, humidity & wind tracking
- 📍 Atmospheric pressure details
- 📱 Fully responsive modern UI

---

## 🧰 Tech Stack

- ⚛️ React.js
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🌐 OpenWeatherMap API
- 🗺️ Google Maps API

---

## 🧠 Core Functionality

This application fetches live weather data from external APIs (such as OpenWeatherMap) and dynamically renders:

- Current temperature
- Weather conditions (sunny, cloudy, rainy, etc.)
- Humidity levels
- Wind speed
- Location-based weather updates

---

## ⚙️ How It Works

1. User searches for a city
2. App sends request to weather API
3. Receives real-time weather data
4. Displays structured weather information
5. Updates map location dynamically

---

## 🧩 Challenges Solved

- Efficient handling of real-time API updates  
- Optimized UI rendering for fast performance  
- Smooth integration of interactive maps
